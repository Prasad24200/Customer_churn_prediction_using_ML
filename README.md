# 🧠 Customer Churn Prediction (ML Project)

Predicts customer churn using the Telco dataset and a Random Forest model.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## How to Run
1. Clone the repo
2. Download the dataset from [Kaggle](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
3. Rename it as `churn.csv` and place it in the repo folder
4. Run the notebook step-by-step

## Output
- Trained model
- Accuracy and classification report
- Example prediction
